News
====

VisionWorkbench is developed as part of the `Ames Stereo Pipeline
<https://stereopipeline.readthedocs.io>`_ (ASP). See the `ASP News
<https://stereopipeline.readthedocs.io/en/latest/news.html>`_ page for
current release notes and development history.

Historical release notes (2006-2011) for standalone VisionWorkbench
releases are archived in the git history of this file.
